## üêõ Common Issues & Fixes

### Popup Not Showing
**Cause:** localStorage already has `emailSubscribed = 'true'`
**Fix:** Open browser console, run: `localStorage.removeItem('emailSubscribed')`

### "emailCollector is not defined" Error
**Cause:** Script not loaded or timing issue
**Fix:** Refresh page, check browser console for errors

### Success Message Doesn't Appear
**Cause:** API connection issue or MAILERLITE_API_KEY not set
**Fix:** Check Vercel environment variables, verify API key is set

### Inline Signup Not Visible
**Cause:** CSS not loaded or element not found
**Fix:** Check browser console, verify `inline-email-signup.css` loads
