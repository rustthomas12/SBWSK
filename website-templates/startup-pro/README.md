## ğŸ¨ Customization Guide

### Changing Colors

Edit `css/main.css` and update these CSS variables:

```css
:root {
    --primary: #0ea5e9;     /* Main brand color */
    --secondary: #8b5cf6;   /* Accent color */
    --text: #1f2937;        /* Text color */
    --background: #ffffff;  /* Background color */
}
```

### Changing Fonts

The template uses system fonts for fast loading. To use Google Fonts:

1. Visit https://fonts.google.com
2. Select your font
3. Add the `<link>` to the `<head>` of each HTML file
4. Update font-family in `css/main.css`

### Adding Pages

1. Duplicate an existing HTML file
2. Update the content
3. Add a link to the navigation menu
4. Keep the same header and footer for consistency
